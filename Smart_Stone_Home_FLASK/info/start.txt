1. venv - установка виртуальной среды
2. pip install flask - установка фреймворка flask
3. pip install flask-sqlalchemy - Flask-SQLAlchemy, расширение, которое обеспечивает Flask-дружественную 
оболочку к популярному пакету SQLAlchemy, который является Object Relational Mapper или ORM. 
ORM позволяют приложениям управлять базой данных с использованием объектов высокого уровня, таких 
как классы, объекты и методы, а не таблицы и SQL. Задача ORM — перевести операции высокого уровня в 
команды базы данных.
4. pip install flask-migrate -  Это расширение является Flask-оберткой для Alembic, 
основой для миграции базы данных SQLAlchemy. Работа с миграциями баз данных добавляет 
немного работы вначале, но это небольшая цена, чтобы заплатить за надежный способ внесения 
изменений в вашу базу данных в будущем.
